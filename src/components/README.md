# Composants

Ce dossier contient les composants réutilisables de l'application.

## Structure recommandée

- `Button/` - Composant bouton
- `Card/` - Composant carte
- `Modal/` - Composant modal
- `Input/` - Composant input
- etc.

Chaque composant peut avoir son propre dossier avec :
- `ComponentName.tsx` - Le composant
- `ComponentName.css` - Les styles (optionnel)
- `ComponentName.test.tsx` - Les tests (optionnel)
